Le Sokoban Autonome (SAÉ 1.02)

C Analyse & Algorithmes Autonomie & Optimisation

🌟 Introduction

Dans ce volet, le projet évolue vers l'automatisation. L'objectif est de transformer le jeu en un interpréteur de scripts capable de lire une suite de mouvements pré-enregistrés (fichier .dep) et de vérifier si elle mène à la victoire sur un plateau donné.

📍 Situation

Le passage à la SAÉ 1.02 impose un changement de paradigme : le programme doit devenir "autonome". Il ne s'agit plus seulement d'écouter le clavier, mais de traiter un flux de données structurées pour "dérouler" une partie sans intervention humaine.

🎯 Task (Mission)

Les exigences pour ce moteur autonome étaient les suivantes :

  • Double chargement : Importer simultanément le plateau (.sok) et la liste des déplacements (.dep).
  • Exécution visuelle : Afficher chaque mouvement à l'écran avec une pause calibrée pour permettre au "client" de suivre l'évolution.
  • Validation algorithmique : Analyser si la séquence fournie est une solution valide ou un échec.
  • Intégration : Fusionner les modes manuel et autonome dans un menu principal unique.

🛠️ Action (Mise en œuvre)

1. Refactoring de l'Architecture

Pour gérer les deux modes, j'ai restructuré le main avec un menu de sélection. La logique de jeu a été encapsulée dans une fonction deplacer polyvalente, capable de fonctionner avec le clavier ou avec un script grâce à un paramètre mode_auto.

2. Contrôle du Temps et Animation

Pour que le replay soit fluide et compréhensible, j'ai implémenté :

  • usleep(250000) : Une pause de 250 000 microsecondes (soit 0,25s) entre chaque mouvement pour simuler une animation à 4 images par seconde.
  • Gestion des transitions : Utilisation de system("clear") combiné à l'affichage de l'entête spécifique au mode autonome (progression X/Y, zoom actuel).

3. Interpréteur de Caractères

Le programme doit traduire les symboles du fichier .dep en actions physiques sur le plateau :

  • Normalisation : Utilisation de tolower() et toupper() pour traiter les directions (h, b, g, d) indépendamment de la casse, tout en distinguant les poussées de caisses.
  • Gestion des métadonnées : Le script peut aussi contenir des instructions de zoom (+, -) ou d'annulation (u), que le moteur traite en temps réel durant le replay.

🏆 Result (Résultat)

Le programme est désormais un véritable outil de test et de validation :

  • Bilan automatique : À la fin du script, le programme affiche un verdict clair : "SOLUTION VALIDE" avec le nombre de coups, ou "ECHEC".
  • Fiabilité : Le moteur détecte immédiatement si une caisse est bloquée ou si le script s'arrête avant la victoire.
  • Ergonomie : L'interface permet de basculer entre le plaisir de jouer (Manuel) et la rigueur de la vérification (Autonome) en un seul clic.

💡 Ce que j'ai appris

Cette version m'a sensibilisé à l'importance de la séparation des préoccupations (Separation of Concerns). En isolant la logique de mouvement du périphérique d'entrée (clavier vs fichier), j'ai pu réutiliser 90% de mon code précédent pour une fonctionnalité totalement différente. C'est un principe fondamental de l'ingénierie logicielle que j'ai pu mettre en pratique concrètement.

void lancer_mode_autonome() {
    t_plateau plateau;
    t_tabDeplacement listeCoups;
    char nomFichierSok[50];
    char nomFichierDep[50];
    int nbCoupsTotal = 0;
    int nbCoupsJoues = 0; 
    int i;
    int ligJoueur, colJoueur;
    int zoom = 1;
    bool victoire = false;
    char coup;
    char direction;

    printf("Nom du fichier partie (.sok) : ");
    scanf("%s", nomFichierSok);
    printf("Nom du fichier deplacements (.dep) : ");
    scanf("%s", nomFichierDep);

    chargerPartie(plateau, nomFichierSok);
    chargerDeplacements(listeCoups, nomFichierDep, &nbCoupsTotal);
    position_joueur(plateau, &ligJoueur, &colJoueur);

    afficher_entete_auto(nomFichierSok, nomFichierDep, 0, nbCoupsTotal, zoom);
    afficher_plateau(plateau, zoom);
    usleep(1000000); // Pause 1s avant départ

    for (i = 0; i < nbCoupsTotal; i++) {
        if (gagne(plateau)) {
            victoire = true;
            break; // Arrêt si gagné avant la fin
        }

        coup = listeCoups[i];

        if (coup == '+') { if (zoom < 3) zoom++; }
        else if (coup == '-') { if (zoom > 1) zoom--; }
        else if (tolower(coup) == 'u') {
             annuler_coup(plateau, listeCoups, &nbCoupsJoues, &ligJoueur, &colJoueur);
        }
        else {
            direction = tolower(coup);
            deplacer(plateau, &ligJoueur, &colJoueur, direction, &nbCoupsJoues, listeCoups, true);
        }

        afficher_entete_auto(nomFichierSok, nomFichierDep, i + 1, nbCoupsTotal, zoom);
        afficher_plateau(plateau, zoom);
        usleep(250000); // Pause 0.25s entre les mouvements
    }

   /* --- Fin de la boucle de jeu --- */

    /* Si on n'a pas détecté de victoire dans la boucle, on vérifie une dernière fois */
    if (!victoire) {
        victoire = gagne(plateau);
    }

    printf("\n--------------------------------------------------\n");
    
    if (victoire) {
        /* Cas où c'est gagné : on affiche le nombre de coups */
        if (i < nbCoupsTotal) {
            /* Si 'i' est plus petit que le total, c'est qu'on a fini avant la fin du fichier */
            printf("SOLUTION VALIDE (Gagne en %d coups).\n", i);
        } else {
            /* Sinon, on a utilisé tous les coups du fichier */
            printf("SOLUTION VALIDE (Gagne en %d coups).\n", nbCoupsTotal);
        }
    } else {
        /* Cas où c'est perdu */
        printf("ECHEC : Partie perdue avec cette liste de coups.\n");
    }

    printf("--------------------------------------------------\n");
}

void lancer_mode_manuel() {
    t_plateau plateau;
    t_tabDeplacement historique;
    char nomFichier[50];
    char nomSauv[50];
    char touche = '\0';
    int ligJoueur, colJoueur;
    int nbCoups = 0;
    int zoom = 1;
    char rep;

    printf("Nom du fichier .sok : ");
    scanf("%s", nomFichier);

    chargerPartie(plateau, nomFichier);
    position_joueur(plateau, &ligJoueur, &colJoueur);

    while (touche != TOUCHE_ABANDON && !gagne(plateau)) {
        afficher_entete_manuel(nomFichier, nbCoups, zoom);
        afficher_plateau(plateau, zoom);

        while (!kbhit()) { usleep(10000); }
        touche = getchar();

        switch (touche) {
            case TOUCHE_RECOMMENCER:
                chargerPartie(plateau, nomFichier);
                position_joueur(plateau, &ligJoueur, &colJoueur);
                nbCoups = 0;
                break;
            case TOUCHE_ZOOM_PLUS: if (zoom < 3) zoom++; break;
            case TOUCHE_ZOOM_MOINS: if (zoom > 1) zoom--; break;
            case TOUCHE_ANNULER: 
                annuler_coup(plateau, historique, &nbCoups, &ligJoueur, &colJoueur);
                break;
            case TOUCHE_ABANDON: break;
            default:
                // false = mode manuel, on enregistre le coup dans l'historique
                deplacer(plateau, &ligJoueur, &colJoueur, touche, &nbCoups, historique, false);
                break;
        }
    }

    afficher_entete_manuel(nomFichier, nbCoups, zoom);
    afficher_plateau(plateau, zoom);

    if (gagne(plateau)) printf("\nBRAVO ! GAGNE !\n");
    else printf("\nABANDON.\n");

    // Sauvegarde Historique (Consigne SAE 1.01V2)
    printf("Sauvegarder l'historique (.dep) ? (o/n) : ");
    while(kbhit()) getchar(); 
    scanf(" %c", &rep);
    if (rep == 'o' || rep == 'O') {
        printf("Nom fichier : ");
        scanf("%s", nomSauv);
        enregistrerDeplacements(historique, nbCoups, nomSauv);
    }
}

void afficher_plateau(t_plateau plateau, int zoom) {
    int i, j, v, h;
    char c, sym;
    for (i = 0; i < TAILLE; i++) {
        for (v = 0; v < zoom; v++) {
            for (j = 0; j < TAILLE; j++) {
                c = plateau[i][j];
                sym = SYM_VIDE;
                if (c == SYM_MUR) sym = SYM_MUR;
                else if (c == SYM_CIBLE) sym = SYM_CIBLE;
                else if (c == SYM_CAISSE || c == SYM_CAISSE_SUR_CIBLE) sym = SYM_CAISSE;
                else if (c == SYM_SOKOBAN || c == SYM_SOKOBAN_SUR_CIBLE) sym = SYM_SOKOBAN;
                for (h = 0; h < zoom; h++) printf("%c", sym);
            }
            printf("\n");
        }
    }
}

bool gagne(t_plateau plateau) {
    int i, j;
    for (i = 0; i < TAILLE; i++)
        for (j = 0; j < TAILLE; j++)
            if (plateau[i][j] == SYM_CAISSE) return false;
    return true;
}

void position_joueur(t_plateau plateau, int *lig, int *col) {
    int i, j;
    for (i = 0; i < TAILLE; i++)
        for (j = 0; j < TAILLE; j++)
            if (plateau[i][j] == SYM_SOKOBAN || plateau[i][j] == SYM_SOKOBAN_SUR_CIBLE) {
                *lig = i; *col = j; return;
            }
}

void deplacer(t_plateau plateau, int *lig, int *col, char direction, 
              int *nbCoups, t_tabDeplacement hist, bool mode_auto) {
    int dY = 0, dX = 0;
    int y = *lig, x = *col;
    int nY, nX, pY, pX;
    char code = '?';
    char *actuel, *suiv, *apres;

    // Switch des deplacements
    switch(tolower(direction)) {
        case 'z': case 'h': dY = -1; code = 'h'; break;
        case 's': case 'b': dY =  1; code = 'b'; break;
        case 'q': case 'g': dX = -1; code = 'g'; break;
        case 'd': dX =  1; code = 'd'; break;
    }
    if (dY == 0 && dX == 0) return;

    nY = y + dY; nX = x + dX;
    if (nY < 0 || nY >= TAILLE || nX < 0 || nX >= TAILLE) return;

    actuel = &plateau[y][x];
    suiv = &plateau[nY][nX];

    // Mouvement Simple
    if (*suiv == SYM_VIDE || *suiv == SYM_CIBLE) {
        *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
        *suiv = (*suiv == SYM_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
        *lig = nY; *col = nX;
        
        // En mode manuel, on enregistre. En auto, on ne fait rien (l'historique est déjà écrit)
        if (!mode_auto && *nbCoups < MAX_HISTORIQUE) hist[(*nbCoups)++] = code;
    }
    // Poussée
    else if (*suiv == SYM_CAISSE || *suiv == SYM_CAISSE_SUR_CIBLE) {
        pY = nY + dY; pX = nX + dX;
        if (pY >= 0 && pY < TAILLE && pX >= 0 && pX < TAILLE) {
            apres = &plateau[pY][pX];
            if (*apres == SYM_VIDE || *apres == SYM_CIBLE) {
                *apres = (*apres == SYM_CIBLE) ? SYM_CAISSE_SUR_CIBLE : SYM_CAISSE;
                *suiv = (*suiv == SYM_CAISSE_SUR_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
                *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
                *lig = nY; *col = nX;

                if (!mode_auto && *nbCoups < MAX_HISTORIQUE) hist[(*nbCoups)++] = toupper(code);
            }
        }
    }
}

void annuler_coup(t_plateau plateau, t_tabDeplacement hist, int *nbCoups, int *lig, int *col) {
    // Note: cette fonction marche pour le mode manuel.
    // Pour le mode auto, il faudrait lui passer le coup exact à annuler car hist n'est pas utilisé pareil.
    if (*nbCoups <= 0) return;
    char coup = hist[*nbCoups - 1];
    int dY=0, dX=0, y=*lig, x=*col, pY, pX, cY, cX;

    switch(tolower(coup)) {
        case 'h': dY = 1; break; 
        case 'b': dY = -1; break;
        case 'g': dX = 1; break;
        case 'd': dX = -1; break;
    }
    pY = y + dY; pX = x + dX;
    
    // Retour Sokoban
    plateau[pY][pX] = (plateau[pY][pX] == SYM_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
    plateau[y][x] = (plateau[y][x] == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;

    // Retour Caisse (si majuscule car signifie poussée)
    if (isupper(coup)) {
        cY = y - dY; cX = x - dX; // Position actuelle caisse (devant)
        // On la remet sur la case (y,x) que le joueur vient de libérer
        plateau[cY][cX] = (plateau[cY][cX] == SYM_CAISSE_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
        plateau[y][x] = (plateau[y][x] == SYM_CIBLE) ? SYM_CAISSE_SUR_CIBLE : SYM_CAISSE;
    }
    *lig = pY; *col = pX;
    (*nbCoups)--;
}

/* --- FONCTIONS FOURNIES --- */
int kbhit(){
    struct termios oldt, newt;
    int ch, oldf;
    tcgetattr(STDIN_FILENO, &oldt);
    newt = oldt; newt.c_lflag &= ~(ICANON | ECHO);
    tcsetattr(STDIN_FILENO, TCSANOW, &newt);
    oldf = fcntl(STDIN_FILENO, F_GETFL, 0);
    fcntl(STDIN_FILENO, F_SETFL, oldf | O_NONBLOCK);
    ch = getchar();
    tcsetattr(STDIN_FILENO, TCSANOW, &oldt);
    fcntl(STDIN_FILENO, F_SETFL, oldf);
    if(ch != EOF){ ungetc(ch, stdin); return 1; } 
    return 0;
}
void chargerPartie(t_plateau plateau, char fichier[]){
    FILE *f = fopen(fichier, "r");
    char fin;
    if(!f){ printf("ERR FICHIER\n"); exit(1); }
    for(int i=0;i<TAILLE;i++){
        for(int j=0;j<TAILLE;j++) fread(&plateau[i][j],1,1,f);
        fread(&fin,1,1,f);
    }
    fclose(f);
}
void enregistrerPartie(t_plateau plateau, char fichier[]){
    FILE *f = fopen(fichier, "w");
    char fin='\n';
    for(int i=0;i<TAILLE;i++){
        for(int j=0;j<TAILLE;j++) fwrite(&plateau[i][j],1,1,f);
        fwrite(&fin,1,1,f);
    }
    fclose(f);
}
void chargerDeplacements(t_tabDeplacement t, char fichier[], int *nb){
    FILE *f = fopen(fichier, "r");
    char c; *nb=0;
    if(!f) { printf("ERR DEP\n"); return; }
    fread(&c,1,1,f);
    while(!feof(f) && *nb < MAX_HISTORIQUE){
        if(c!='\n' && c!='\r') t[(*nb)++] = c;
        fread(&c,1,1,f);
    }
    fclose(f);
}
void enregistrerDeplacements(t_tabDeplacement t, int nb, char fic[]){
    FILE *f = fopen(fic, "w");
    fwrite(t, 1, nb, f);
    fclose(f);
}

🌟 Introduction

La dernière étape de ce projet de première année consiste à s'attaquer à l'efficience algorithmique. L'objectif n'est plus seulement de trouver une solution, mais de produire la solution la plus courte possible en éliminant le "bruit" (mouvements inutiles) d'une séquence de jeu pré-enregistrée.

📍 Situation

Dans le cadre de la SAÉ 1.02, nous devions traiter des fichiers de déplacements (.dep) parfois mal optimisés (erreurs de parcours, collisions contre les murs, boucles inutiles). Le programme doit agir comme un compilateur : il lit un code source (les mouvements) et produit un code optimisé plus performant.

🎯 Task (Mission)

Ma mission consistait à implémenter deux types de nettoyage automatique :

  • Suppression des mouvements non joués : Éliminer tout caractère qui ne provoque pas de déplacement réel (frapper un mur).
  • Élimination des séquences inutiles : Détecter et supprimer les boucles où le joueur revient à son point de départ sans avoir déplacé de caisse entre-temps.
  • Comparaison de performance : Afficher la longueur initiale vs la longueur optimisée et proposer l'enregistrement du résultat.

🛠️ Action (Mise en œuvre)

1. Détection de Cycles et Backtracking

Pour supprimer les boucles, j'ai implémenté une structure de données t_positionHist capable de mémoriser les coordonnées (x, y) du joueur associées à leur index dans l'historique.

  • Algorithme : À chaque pas, le programme vérifie si la position actuelle a déjà été visitée depuis la dernière poussée de caisse.
  • Élagage : Si une position est redondante, le programme effectue un "rollback" (retour arrière) immédiat dans l'historique des coups, supprimant ainsi la boucle inutile avant même qu'elle ne soit affichée.

2. Calcul des Déplacements "Nets"

J'ai développé une fonction calculerCoupsOptimises qui analyse les segments de déplacements simples :

  • Elle calcule la distance verticale et horizontale minimale nécessaire pour atteindre un point de poussée.
  • Si un joueur fait des détours en "S" alors qu'une ligne droite est possible, l'algorithme réduit le segment à son expression la plus simple.

3. Gestion Dynamique de la Grille

Contrairement aux versions précédentes limitées à du 12×12, cette version gère des grilles de tailles variables jusqu'à 50×50 grâce à une détection automatique lors du chargement (gHauteur et gLargeur).

🏆 Result (Résultat)

Le programme final est un optimiseur de trajectoire performant :

  • Réduction drastique : Sur des fichiers de test complexes (ex: 999 caractères), le programme parvient souvent à réduire la séquence de 20% à 50% en supprimant les hésitations du joueur.
  • Validation de Solution : Le programme confirme systématiquement si la séquence finale permet toujours de gagner la partie.
  • Professionnalisme : L'utilisateur peut sauvegarder son "score" optimisé dans un nouveau fichier .dep pour le partager.

💡 Ce que j'ai appris

Ce projet final a été une introduction concrète à l'optimisation de chemin (Pathfinding). J'ai appris que l'informatique ne sert pas seulement à résoudre un problème, mais à le résoudre avec élégance et économie de ressources. La gestion de structures complexes comme t_tabPositions m'a permis de mieux comprendre comment les logiciels de CAO ou les moteurs de jeux gèrent l'historique des actions (Undo/Redo et optimisation).

#define SYM_CAISSE_SUR_CIBLE '*'

#define SYM_SOKOBAN_SUR_CIBLE '+'

/* Fonction pour calculer le nombre de coups optimisés à partir de l'historique
   L'optimisation des déplacements nets ne s'applique que s'il y a une seule poussée */
int calculerCoupsOptimises(t_tabDeplacement hist, int nbCoups) {
  int nbPoussees = 0;
  int i;
  char c;

  // Compter le nombre de poussées
  for (i = 0; i < nbCoups; i++) {
    if (isupper(hist[i]))
      nbPoussees++;
  }

  // Si plus d'une poussée, pas d'optimisation des déplacements nets
  if (nbPoussees > 1) {
    return nbCoups;
  }

  // Une seule poussée (ou zéro) : appliquer l'optimisation des déplacements nets
  int total = 0;
  int cptHaut = 0, cptBas = 0, cptGauche = 0, cptDroite = 0;

  for (i = 0; i < nbCoups; i++) {
    c = hist[i];
    if (isupper(c)) {
      // C'est une poussée - calculer les déplacements nets du segment actuel
      int netsV = (cptHaut > cptBas) ? (cptHaut - cptBas) : (cptBas - cptHaut);
      int netsH = (cptDroite > cptGauche) ? (cptDroite - cptGauche) : (cptGauche - cptDroite);
      total += netsV + netsH + 1; // +1 pour la poussée
      // Réinitialiser les compteurs
      cptHaut = 0;
      cptBas = 0;
      cptGauche = 0;
      cptDroite = 0;
    } else {
      // Déplacement simple
      if (c == 'h')
        cptHaut++;
      else if (c == 'b')
        cptBas++;
      else if (c == 'g')
        cptGauche++;
      else if (c == 'd')
        cptDroite++;
    }
  }
  // Ajouter les déplacements nets du dernier segment (après la dernière poussée)
  int netsV = (cptHaut > cptBas) ? (cptHaut - cptBas) : (cptBas - cptHaut);
  int netsH = (cptDroite > cptGauche) ? (cptDroite - cptGauche) : (cptGauche - cptDroite);
  total += netsV + netsH;

  return total;
}

void lancer_mode_autonome() {
  t_plateau plateau;
  t_tabDeplacement listeCoups;
  t_tabDeplacement historique; // Historique des coups réellement effectués
  t_tabPositions positions;    // Historique des positions pour optimisation
  char nomFichierSok[50];
  char nomFichierDep[50];
  int nbCoupsTotal = 0;
  int nbCoupsJoues = 0; // Nombre de coups dans l'historique (après undo)
  int nbPositions = 0;  // Nombre de positions mémorisées
  int dernierePousseIndex = -1; // Index du dernier coup de poussée
  int i;
  int ligJoueur, colJoueur;
  int zoom = 1;
  bool victoire = false;
  char coup;
  bool estPoussee;

  printf("Nom du fichier partie (.sok) : ");
  scanf("%s", nomFichierSok);
  printf("Nom du fichier deplacements (.dep) : ");
  scanf("%s", nomFichierDep);

  chargerPartie(plateau, nomFichierSok);
  chargerDeplacements(listeCoups, nomFichierDep, &nbCoupsTotal);
  position_joueur(plateau, &ligJoueur, &colJoueur);

  // Mémoriser la position initiale
  positions[0].lig = ligJoueur;
  positions[0].col = colJoueur;
  positions[0].indexHist = 0;
  nbPositions = 1;

  // Vérifier si la partie est déjà gagnée au départ
  if (gagne(plateau)) {
    afficher_entete_auto(nomFichierSok, nomFichierDep, 0, nbCoupsTotal, zoom);
    afficher_plateau(plateau, zoom);
    printf("\n--------------------------------------------------\n");
    printf("SOLUTION VALIDE (Gagne en 0 coups).\n");
    printf("--------------------------------------------------\n");
    return;
  }

  afficher_entete_auto(nomFichierSok, nomFichierDep, 0, nbCoupsTotal, zoom);
  afficher_plateau(plateau, zoom);
  usleep(1000000); // Pause 1s avant départ

  for (i = 0; i < nbCoupsTotal; i++) {
    if (gagne(plateau)) {
      victoire = true;
      break; // Arrêt si gagné avant la fin
    }

    coup = listeCoups[i];

    if (coup == '+') {
      if (zoom < 3)
        zoom++;
    } else if (coup == '-') {
      if (zoom > 1)
        zoom--;
    } else if (tolower(coup) == 'u') {
      // Annuler en utilisant l'historique des coups réellement effectués
      annuler_coup(plateau, historique, &nbCoupsJoues, &ligJoueur, &colJoueur);
    } else {
      // Vérifier si c'est une poussée demandée (majuscule)
      estPoussee = isupper(coup);
      // Effectuer le déplacement et enregistrer dans l'historique
      deplacer_auto(plateau, &ligJoueur, &colJoueur, coup, &nbCoupsJoues,
                    historique, estPoussee, positions, &nbPositions,
                    &dernierePousseIndex);
    }

    afficher_entete_auto(nomFichierSok, nomFichierDep, i + 1, nbCoupsTotal,
                         zoom);
    afficher_plateau(plateau, zoom);
    usleep(250000); // Pause 0.25s entre les mouvements
  }

  /* --- Fin de la boucle de jeu --- */

  /* Si on n'a pas détecté de victoire dans la boucle, on vérifie une dernière
   * fois */
  if (!victoire) {
    victoire = gagne(plateau);
  }

  printf("\n--------------------------------------------------\n");

  /* Calculer le nombre de coups optimisés :
     - L'historique a déjà été nettoyé des boucles par deplacer_auto
     - calculerCoupsOptimises calcule ensuite les déplacements nets par segment */
  int coupsOptimises = calculerCoupsOptimises(historique, nbCoupsJoues);

  if (victoire) {
    printf("SOLUTION VALIDE (Gagne en %d coups).\n", coupsOptimises);
  } else {
    /* Partie non gagnée */
    if (coupsOptimises > 0) {
      printf("Partie non gagnée. Coups effectués: %d\n", coupsOptimises);
    } else {
      printf("ECHEC : Partie perdue avec cette liste de coups.\n");
    }
  }

  printf("--------------------------------------------------\n");
}

void lancer_mode_manuel() {
  t_plateau plateau;
  t_tabDeplacement historique;
  char nomFichier[50];
  char nomSauv[50];
  char touche = '\0';
  int ligJoueur, colJoueur;
  int nbCoups = 0;
  int zoom = 1;
  char rep;

  printf("Nom du fichier .sok : ");
  scanf("%s", nomFichier);

  chargerPartie(plateau, nomFichier);
  position_joueur(plateau, &ligJoueur, &colJoueur);

  while (touche != TOUCHE_ABANDON && !gagne(plateau)) {
    afficher_entete_manuel(nomFichier, nbCoups, zoom);
    afficher_plateau(plateau, zoom);

    while (!kbhit()) {
      usleep(10000);
    }
    touche = getchar();

    switch (touche) {
    case TOUCHE_RECOMMENCER:
      chargerPartie(plateau, nomFichier);
      position_joueur(plateau, &ligJoueur, &colJoueur);
      nbCoups = 0;
      break;
    case TOUCHE_ZOOM_PLUS:
      if (zoom < 3)
        zoom++;
      break;
    case TOUCHE_ZOOM_MOINS:
      if (zoom > 1)
        zoom--;
      break;
    case TOUCHE_ANNULER:
      annuler_coup(plateau, historique, &nbCoups, &ligJoueur, &colJoueur);
      break;
    case TOUCHE_ABANDON:
      break;
    default:
      // false = mode manuel, on enregistre le coup dans l'historique
      deplacer(plateau, &ligJoueur, &colJoueur, touche, &nbCoups, historique,
               false);
      break;
    }
  }

  afficher_entete_manuel(nomFichier, nbCoups, zoom);
  afficher_plateau(plateau, zoom);

  if (gagne(plateau))
    printf("\nBRAVO ! GAGNE !\n");
  else
    printf("\nABANDON.\n");

  // Sauvegarde Historique (Consigne SAE 1.01V2)
  printf("Sauvegarder l'historique (.dep) ? (o/n) : ");
  while (kbhit())
    getchar();
  scanf(" %c", &rep);
  if (rep == 'o' || rep == 'O') {
    printf("Nom fichier : ");
    scanf("%s", nomSauv);
    enregistrerDeplacements(historique, nbCoups, nomSauv);
  }
}

void afficher_plateau(t_plateau plateau, int zoom) {
  int i, j, v, h;
  char c, sym;
  for (i = 0; i < gHauteur; i++) {
    for (v = 0; v < zoom; v++) {
      for (j = 0; j < gLargeur; j++) {
        c = plateau[i][j];
        sym = SYM_VIDE;
        if (c == SYM_MUR)
          sym = SYM_MUR;
        else if (c == SYM_CIBLE)
          sym = SYM_CIBLE;
        else if (c == SYM_CAISSE || c == SYM_CAISSE_SUR_CIBLE)
          sym = SYM_CAISSE;
        else if (c == SYM_SOKOBAN || c == SYM_SOKOBAN_SUR_CIBLE)
          sym = SYM_SOKOBAN;
        for (h = 0; h < zoom; h++)
          printf("%c", sym);
      }
      printf("\n");
    }
  }
}

bool gagne(t_plateau plateau) {
  int i, j;
  for (i = 0; i < gHauteur; i++)
    for (j = 0; j < gLargeur; j++)
      if (plateau[i][j] == SYM_CAISSE)
        return false;
  return true;
}

void position_joueur(t_plateau plateau, int *lig, int *col) {
  int i, j;
  for (i = 0; i < gHauteur; i++)
    for (j = 0; j < gLargeur; j++)
      if (plateau[i][j] == SYM_SOKOBAN ||
          plateau[i][j] == SYM_SOKOBAN_SUR_CIBLE) {
        *lig = i;
        *col = j;
        return;
      }
}

void deplacer(t_plateau plateau, int *lig, int *col, char direction,
              int *nbCoups, t_tabDeplacement hist, bool mode_auto) {
  int dY = 0, dX = 0;
  int y = *lig, x = *col;
  int nY, nX, pY, pX;
  char code = '?';
  char *actuel, *suiv, *apres;

  // Switch des deplacements
  switch (tolower(direction)) {
  case 'z':
  case 'h':
    dY = -1;
    code = 'h';
    break;
  case 's':
  case 'b':
    dY = 1;
    code = 'b';
    break;
  case 'q':
  case 'g':
    dX = -1;
    code = 'g';
    break;
  case 'd':
    dX = 1;
    code = 'd';
    break;
  }
  if (dY == 0 && dX == 0)
    return;

  nY = y + dY;
  nX = x + dX;
  if (nY < 0 || nY >= gHauteur || nX < 0 || nX >= gLargeur)
    return;

  actuel = &plateau[y][x];
  suiv = &plateau[nY][nX];

  // Mouvement Simple
  if (*suiv == SYM_VIDE || *suiv == SYM_CIBLE) {
    *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
    *suiv = (*suiv == SYM_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
    *lig = nY;
    *col = nX;

    // En mode manuel, on enregistre. En auto, on ne fait rien (l'historique est
    // déjà écrit)
    if (!mode_auto && *nbCoups < MAX_HISTORIQUE)
      hist[(*nbCoups)++] = code;
  }
  // Poussée
  else if (*suiv == SYM_CAISSE || *suiv == SYM_CAISSE_SUR_CIBLE) {
    pY = nY + dY;
    pX = nX + dX;
    if (pY >= 0 && pY < gHauteur && pX >= 0 && pX < gLargeur) {
      apres = &plateau[pY][pX];
      if (*apres == SYM_VIDE || *apres == SYM_CIBLE) {
        *apres = (*apres == SYM_CIBLE) ? SYM_CAISSE_SUR_CIBLE : SYM_CAISSE;
        *suiv = (*suiv == SYM_CAISSE_SUR_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE
                                                : SYM_SOKOBAN;
        *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
        *lig = nY;
        *col = nX;

        if (!mode_auto && *nbCoups < MAX_HISTORIQUE)
          hist[(*nbCoups)++] = toupper(code);
      }
    }
  }
}

void deplacer_auto(t_plateau plateau, int *lig, int *col, char direction,
                   int *nbCoups, t_tabDeplacement hist, bool estPoussee,
                   t_tabPositions positions, int *nbPositions,
                   int *dernierePousseIndex) {
  int dY = 0, dX = 0;
  int y = *lig, x = *col;
  int nY, nX, pY, pX;
  char code = '?';
  char *actuel, *suiv, *apres;
  int i;

  // Switch des deplacements
  switch (tolower(direction)) {
  case 'z':
  case 'h':
    dY = -1;
    code = 'h';
    break;
  case 's':
  case 'b':
    dY = 1;
    code = 'b';
    break;
  case 'q':
  case 'g':
    dX = -1;
    code = 'g';
    break;
  case 'd':
    dX = 1;
    code = 'd';
    break;
  }
  if (dY == 0 && dX == 0)
    return;

  nY = y + dY;
  nX = x + dX;
  if (nY < 0 || nY >= gHauteur || nX < 0 || nX >= gLargeur)
    return;

  actuel = &plateau[y][x];
  suiv = &plateau[nY][nX];

  // Si c'est une poussée demandée (majuscule), on doit pousser une caisse
  if (estPoussee) {
    if (*suiv == SYM_CAISSE || *suiv == SYM_CAISSE_SUR_CIBLE) {
      pY = nY + dY;
      pX = nX + dX;
      if (pY >= 0 && pY < gHauteur && pX >= 0 && pX < gLargeur) {
        apres = &plateau[pY][pX];
        if (*apres == SYM_VIDE || *apres == SYM_CIBLE) {
          *apres = (*apres == SYM_CIBLE) ? SYM_CAISSE_SUR_CIBLE : SYM_CAISSE;
          *suiv = (*suiv == SYM_CAISSE_SUR_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE
                                                  : SYM_SOKOBAN;
          *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
          *lig = nY;
          *col = nX;

          if (*nbCoups < MAX_HISTORIQUE)
            hist[(*nbCoups)++] = toupper(code);

          // Après une poussée, on réinitialise les positions car on ne peut
          // plus optimiser les mouvements précédents
          *dernierePousseIndex = *nbCoups;
          positions[0].lig = nY;
          positions[0].col = nX;
          positions[0].indexHist = *nbCoups;
          *nbPositions = 1;
        }
      }
    }
    // Si pas de caisse à pousser, le coup est ignoré
  } else {
    // Mouvement simple (minuscule) - ne doit pas pousser de caisse
    if (*suiv == SYM_VIDE || *suiv == SYM_CIBLE) {
      *actuel = (*actuel == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
      *suiv = (*suiv == SYM_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
      *lig = nY;
      *col = nX;

      if (*nbCoups < MAX_HISTORIQUE)
        hist[(*nbCoups)++] = code;

      // Vérifier si on est déjà passé par cette position (optimisation)
      for (i = 0; i < *nbPositions; i++) {
        if (positions[i].lig == nY && positions[i].col == nX) {
          // On est revenu à une position déjà visitée !
          // On annule tous les coups depuis cette position
          *nbCoups = positions[i].indexHist;
          *nbPositions = i + 1; // On garde les positions jusqu'à celle-ci
          return;
        }
      }

      // Mémoriser la nouvelle position
      if (*nbPositions < MAX_HISTORIQUE) {
        positions[*nbPositions].lig = nY;
        positions[*nbPositions].col = nX;
        positions[*nbPositions].indexHist = *nbCoups;
        (*nbPositions)++;
      }
    }
    // Si c'est une caisse ou un mur, le coup est ignoré
  }
}

void annuler_coup(t_plateau plateau, t_tabDeplacement hist, int *nbCoups,
                  int *lig, int *col) {
  // Note: cette fonction marche pour le mode manuel.
  // Pour le mode auto, il faudrait lui passer le coup exact à annuler car hist
  // n'est pas utilisé pareil.
  if (*nbCoups <= 0)
    return;
  char coup = hist[*nbCoups - 1];
  int dY = 0, dX = 0, y = *lig, x = *col, pY, pX, cY, cX;

  switch (tolower(coup)) {
  case 'h':
    dY = 1;
    break;
  case 'b':
    dY = -1;
    break;
  case 'g':
    dX = 1;
    break;
  case 'd':
    dX = -1;
    break;
  }
  pY = y + dY;
  pX = x + dX;

  // Retour Sokoban
  plateau[pY][pX] =
      (plateau[pY][pX] == SYM_CIBLE) ? SYM_SOKOBAN_SUR_CIBLE : SYM_SOKOBAN;
  plateau[y][x] =
      (plateau[y][x] == SYM_SOKOBAN_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;

  // Retour Caisse (si majuscule car signifie poussée)
  if (isupper(coup)) {
    cY = y - dY;
    cX = x - dX; // Position actuelle caisse (devant)
    // On la remet sur la case (y,x) que le joueur vient de libérer
    plateau[cY][cX] =
        (plateau[cY][cX] == SYM_CAISSE_SUR_CIBLE) ? SYM_CIBLE : SYM_VIDE;
    plateau[y][x] =
        (plateau[y][x] == SYM_CIBLE) ? SYM_CAISSE_SUR_CIBLE : SYM_CAISSE;
  }
  *lig = pY;
  *col = pX;
  (*nbCoups)--;
}

/* --- FONCTIONS FOURNIES --- */
int kbhit() {
  struct termios oldt, newt;
  int ch, oldf;
  tcgetattr(STDIN_FILENO, &oldt);
  newt = oldt;
  newt.c_lflag &= ~(ICANON | ECHO);
  tcsetattr(STDIN_FILENO, TCSANOW, &newt);
  oldf = fcntl(STDIN_FILENO, F_GETFL, 0);
  fcntl(STDIN_FILENO, F_SETFL, oldf | O_NONBLOCK);
  ch = getchar();
  tcsetattr(STDIN_FILENO, TCSANOW, &oldt);
  fcntl(STDIN_FILENO, F_SETFL, oldf);
  if (ch != EOF) {
    ungetc(ch, stdin);
    return 1;
  }
  return 0;
}
void chargerPartie(t_plateau plateau, char fichier[]) {
  FILE *f = fopen(fichier, "r");
  char c;
  int ligne = 0, col = 0;

  if (!f) {
    printf("ERR FICHIER\n");
    exit(1);
  }

  // Initialiser le plateau avec des espaces
  for (int i = 0; i < TAILLE_MAX; i++)
    for (int j = 0; j < TAILLE_MAX; j++)
      plateau[i][j] = SYM_VIDE;

  // Lire le fichier caractère par caractère pour détecter la taille
  gHauteur = 0;
  gLargeur = 0;

  while (fread(&c, 1, 1, f) == 1 && ligne < TAILLE_MAX) {
    if (c == '\n' || c == '\r') {
      if (col > 0) { // Ignorer les lignes vides
        if (col > gLargeur)
          gLargeur = col;
        ligne++;
        col = 0;
      }
    } else {
      if (col < TAILLE_MAX) {
        plateau[ligne][col] = c;
        col++;
      }
    }
  }
  // Gérer la dernière ligne si pas de newline à la fin
  if (col > 0) {
    if (col > gLargeur)
      gLargeur = col;
    ligne++;
  }
  gHauteur = ligne;

  fclose(f);
}
void enregistrerPartie(t_plateau plateau, char fichier[]) {
  FILE *f = fopen(fichier, "w");
  char fin = '\n';
  for (int i = 0; i < gHauteur; i++) {
    for (int j = 0; j < gLargeur; j++)
      fwrite(&plateau[i][j], 1, 1, f);
    fwrite(&fin, 1, 1, f);
  }
  fclose(f);
}
void chargerDeplacements(t_tabDeplacement t, char fichier[], int *nb) {
  FILE *f = fopen(fichier, "r");
  char c;
  *nb = 0;
  if (!f) {
    printf("ERR DEP\n");
    return;
  }
  fread(&c, 1, 1, f);
  while (!feof(f) && *nb < MAX_HISTORIQUE) {
    if (c != '\n' && c != '\r')
      t[(*nb)++] = c;
    fread(&c, 1, 1, f);
  }
  fclose(f);
}
void enregistrerDeplacements(t_tabDeplacement t, int nb, char fic[]) {
  FILE *f = fopen(fic, "w");
  fwrite(t, 1, nb, f);
  fclose(f);
}
